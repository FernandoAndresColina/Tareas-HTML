Ejemplo de uso de entidades

En este ejemplo vamos a explicar cómo se utilizaría las entidades por ejemplo cuando queremos explicar cómo poner un párrafo en html. Para poner un párrafo tendríamos que usar una etiqueta de apertura <p>aquí iría el contenido del párrafo y cuando termines tendréis que usar una etiqueta de cierre como esta</p>.
